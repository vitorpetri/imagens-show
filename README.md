## Instalação

Para ter uma cópia local instalada e funcionando, siga estas etapas simples.

1. Clonar o repositório
   ```sh
   git clone https://github.com/vitorpetri/galeria-imagens-show.git
   ```
2. Iniciar o Sass
   ```sh
   npm run sass
   ```
